SPACEMANIA BETA

== IMPORTANTE ==
- El juego solo cuenta con la funcionalidad del primer nivel.
- Cuando el jugador pierde, debería mostrarse una pantalla de "Game Over"; cuando el jugador gana, debería mostrarse una pantalla de "Victory". En esta versión del juego, se volverá a la pantalla de "Level Select" si ocurre alguna de estas dos posibilidades.

== CÓMO JUGAR ==
- Para probar el nivel:
  1. Una vez se abra el juego, dar click en "New Game".
  2. Dar click en alguno de los personajes acompañantes, y dar click en "Confirm". En esta vesión del juego, no importa cúal se escoga.
  3. Dar click en "Play Level".

- Controles (no se recomienda jugar con trackpad)
  - Mouse para apuntar.
  - Click izquierdo para disparar (se puede dejar presionado)
  - WASD para mover al jugador.
  - Barra de espacio para hacer "dash" a una dirección.

- Existen tres elementos de UI importantes al comenzar el nivel:
  - El tipo de bala, que en esta versión no se puede cambiar.
  - La "armadura" (barra roja). Si ésta llega a cero, pierdes el juego.
  - La "energía" (barra amarilla). Hacer "dashes" consume energía.

== AYUDA ==
- El objetivo del primer nivel es destruir a los enemigos hasta que aparezca el "boss". Una vez se destruye al "boss", se pasa el nivel.
- Al destruir un enemigo, ocasionalmente aparecerá un "powerup" que permite regenerar armadura del jugador.
- Cuando el jugador se encuentra haciendo un "dash", es completamente invulnerable a cualquier disparo de un enemigo. Se debe tomar esto en cuenta para poder completar el primer nivel.
